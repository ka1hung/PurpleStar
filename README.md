# 紫微星盤 Purple Star Astrology

免費、開源的紫微斗數命盤計算工具，支援 AI 命理諮詢。

**線上體驗**: https://ka1hung.github.io/PurpleStar/

## 核心特色

- **真太陽時校正** - 依出生地經度自動校正時辰，還原古法精準度
- **8 種 AI 命理師** - 從傳統道長到台式神婆，多種風格任選
- **完整科普教學** - 新手也能看懂的紫微斗數入門指南
- **三語系支援** - 繁體中文、簡體中文、English
- **純本地運算** - 無需註冊，資料不上傳，隱私有保障

## 技術棧

- React 19 + TypeScript
- Tailwind CSS
- Vite
- i18next (多語系)
- Zustand (狀態管理)
- lunar-javascript (農曆轉換)

## 開發

```bash
# 安裝依賴
npm install --legacy-peer-deps

# 啟動開發伺服器
npm run dev

# 建構生產版本
npm run build
```

## 部署

本專案使用 GitHub Pages 部署。推送到 `main` 分支時會自動觸發部署。

## 命盤計算

- 採用三合派計算規則
- 子時換日：23:00 後算隔天
- 閏月處理：閏月視同當月
- 支援真太陽時校正
