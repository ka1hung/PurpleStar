# 紫微斗數命盤計算系統 - 開發計畫

## 專案概述

建立一個純前端的紫微斗數命盤計算應用，透過出生年月日時與性別計算命盤，並整合 OpenAI API 提供命理諮詢功能。

### 核心特色亮點

| 特色 | 說明 |
|------|------|
| **真太陽時校正** | 依出生地經度自動校正時辰，還原古法精準度 |
| **多風格 AI 命理師** | 8 種不同解盤風格，從傳統道長到台式神婆任選 |
| **完整科普教學** | 新手也能看懂的紫微斗數入門指南 |
| **中國古典風 UI** | 精緻的傳統視覺設計，沉浸式體驗 |
| **三語系支援** | 繁中、簡中、英文自由切換 |
| **純本地運算** | 無需註冊，資料不上傳，隱私有保障 |

## 核心功能模組

### 1. 命盤輸入模組
- 西元年月日選擇器
- 出生時間（時:分，精確輸入）
- **出生地點選擇**（核心功能，用於真太陽時校正）
  - 城市搜尋選擇器（內建華人地區主要城市）
  - 或手動輸入經度
  - 顯示時區與經度資訊
- 性別選擇（男/女，影響大限順逆）
- 農曆/西曆轉換（需引入農曆轉換庫）
- **真太陽時校正結果顯示**
  - 標準時間 vs 真太陽時對比
  - 校正後的時辰（子、丑、寅...）
  - 若跨時辰則特別提醒
- 輸入驗證與錯誤提示

### 2. 命盤計算引擎

**採用規則（主流三合派）**：
| 規則 | 採用方式 | 說明 |
|------|---------|------|
| 子時換日 | 23:00 後算隔天 | 23:00-00:59 的子時，日期算隔天 |
| 閏月處理 | 閏月視同當月 | 閏四月 → 當四月計算 |
| 計算派別 | 三合派 | 最主流，90% 線上命盤採用 |

**星曜安置**：
- 農曆轉換（西曆 → 農曆）
- 五行局計算（水二局、木三局、金四局、土五局、火六局）
- 命宮、身宮定位
- 十二宮位排列
- **十四主星**：紫微、天機、太陽、武曲、天同、廉貞、天府、太陰、貪狼、巨門、天相、天梁、七殺、破軍
- **六吉星**：文昌、文曲、左輔、右弼、天魁、天鉞
- **六煞星**：擎羊、陀羅、火星、鈴星、地空、地劫
- **四化星**：化祿、化權、化科、化忌
- **雜曜**：天刑、天姚、紅鸞、天喜、咸池、天馬、祿存、天空、旬空、截空、龍池、鳳閣、天哭、天虛、孤辰、寡宿、蜚廉、破碎、華蓋、天德、月德、天官、天福、解神、天巫、天月、陰煞、三台、八座、恩光、天貴、台輔、封誥、天廚
- 大限、流年計算

### 3. 命盤顯示模組
- 十二宮格視覺化呈現
- 星曜顏色區分（主星、輔星、煞星不同顏色）
- 宮位名稱與星曜清晰排版
- 中央區域顯示基本資料（姓名、生辰、五行局等）

### 4. AI 諮詢聊天室
- OpenAI API 整合（GPT-4 / GPT-3.5）
- API Key 本地安全儲存
- 命盤資料自動注入對話上下文
- 聊天記錄保存
- Markdown 格式支援
- Streaming 串流回應
- **多風格命理師選擇**（見下方詳細說明）

### 真太陽時校正（核心特色功能）

**這是本系統的主打特色！大多數線上命盤工具都忽略了這個關鍵細節。**

#### 什麼是真太陽時？

古代中國計時依據太陽實際位置（日晷），稱為「真太陽時」或「地方時」。現代使用的是統一時區的「標準時間」，與真太陽時存在差異。

#### 為什麼很重要？

**時辰決定命宮，差一個時辰整盤皆變！**

許多人算命多年卻覺得「不太準」，很可能就是因為時辰沒有校正。

#### 差異來源

1. **經度差**：同一時區內，東西方日出時間不同
   - 例：台北（東經 121°）與烏魯木齊（東經 87°）同屬 UTC+8
   - 但實際日出時間差約 **2 小時 16 分鐘**！
   - 烏魯木齊早上 8 點出生，真太陽時其實是約 5:44

2. **均時差**：地球軌道橢圓造成的季節性偏差（±15 分鐘）
   - 11 月快約 16 分鐘，2 月慢約 14 分鐘

#### 實作方式

- **預設啟用**真太陽時校正
- 使用者輸入出生地（城市選擇或經度輸入）
- 內建主要城市經度資料庫（台灣、中國、香港、新加坡、馬來西亞等華人地區）
- 系統自動計算：經度時差 + 均時差
- 顯示校正前後的時辰對比
- 可選擇關閉（標準時間模式）

#### 宣傳文案

> **「您的命盤，可能一直都算錯了時辰」**
>
> 紫微星盤是少數採用「真太陽時」校正的命盤工具。
> 古人用日晷計時，現代人用時區標準時間——這個差異可能讓您的命宮偏移一整個時辰。
>
> 輸入您的出生地，讓我們還原最準確的時辰，給您一張真正屬於您的命盤。
